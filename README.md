# SakiSky Mod (中国版光遇辅助工具)

> 光遇辅助工具

**SakiSky Mod** 是一款面向《光·遇》中国服的模组注入工具，提供变速、自动跑图、无限能量、坐标传送、地图传送等一系列辅助功能，帮助玩家更高效地探索天空王国。

(不再开源以及免费使用，有需求者请前往"[https://m.tb.cn/h.Ry5nLLG?tk=6L1ggN2mxER](https://m.tb.cn/h.RALsfKP?tk=Ol9UgpymIbD)")

---

## 主要功能

### ⚡ 变速
自由调节游戏运行速度，支持加速与减速，灵活控制游戏节奏。

### 🏃 自动跑图
每日任务自动完成，智能寻路跑图，解放双手。

### ✨ 无限能量
解除飞行能量限制，实现无限飞行，自由探索光之国度。

### 📍 坐标传送
精准输入目标坐标，一键瞬移至地图任意位置。

### 🗺️ 地图传送
支持多张地图双击直达。

### 🕯️ 更多辅助
- 自动光翼收集
- 自动献祭（暴风眼）
- 原地光翼

## 安装与使用

1. 下载 `SakiSky Mod` 主程序
2. 启动《光·遇》并进入游戏
3. 运行 SakiSky Mod，点击 **一键注入**
4. 注入成功后，在工具界面选择需要的功能模块
5. 根据需要勾选功能开关，即可生效

> **注意**：本工具仅供技术学习研究使用。使用完毕后请在 24 小时内从所有设备中永久删除。
